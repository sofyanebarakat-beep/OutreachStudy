⚙️ GA4 Connection Setup

Connect your Google Analytics 4 property to load real data

🔗

Connect Google Analytics 4

Enter your GA4 Property ID and Google OAuth Client ID to load real data automatically. You only need to do this once.

Step 1
Go to console.cloud.google.com
Create/select a project → Enable Google Analytics Data API
Step 2
APIs & Services → Credentials → Create OAuth 2.0 Client ID (Web app)
Add http://localhost:8080 as authorized origin
Step 3
Copy your Client ID and your GA4 Property ID from GA4 Admin → Property Settings
→ Google Cloud Console → Google Analytics 4 → GA4 Data API Docs

📍 Where to find your Property ID

  • Open analytics.google.com
  • Click ⚙️ Admin (bottom left)
  • Under "Property" column → Property Settings
  • Copy the Property ID (numbers only, e.g. 12345678)
  • Format it as: properties/12345678

🔑 OAuth Client ID setup

  • Go to console.cloud.google.com
  • Select your project (or create one)
  • APIs & Services → Library → Search "Analytics Data API" → Enable
  • Credentials → + Create Credentials → OAuth Client ID
  • Type: Web application
  • Authorized JS origins: http://localhost:8080
  • Copy the Client ID

🔒 Privacy & Security

Your credentials are saved only in your browser's localStorage. No data is sent to any third-party server. The OAuth token is used only to call the GA4 Data API directly from your browser.


To use on a different machine, repeat the setup. To revoke access, sign out from the topbar.
